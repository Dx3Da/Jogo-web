
# 🎮 Jogo Web

Projeto desenvolvido como parte de um trabalho acadêmico da faculdade, com foco na criação de uma aplicação web interativa utilizando HTML, CSS, JavaScript e Python.

> ⚠️ Este projeto foi desenvolvido em um curto período de tempo para atender aos requisitos da disciplina. O foco principal foi entregar um produto funcional, priorizando a implementação das funcionalidades em vez da arquitetura do código.

---

# 📖 Sobre o Projeto

O projeto consiste em um jogo web onde o usuário pode interagir com diferentes elementos do sistema, explorando mecânicas de gerenciamento e progressão.

Durante o desenvolvimento foram aplicados conceitos como:

- Manipulação do DOM
- Eventos JavaScript
- Atualização dinâmica da interface
- Organização da lógica do jogo
- Integração com Python para persistência de dados
- Estruturação de páginas utilizando HTML e CSS

---

# 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Python

---

# 🤖 Utilização de Inteligência Artificial

Durante o desenvolvimento utilizei ferramentas de IA para acelerar a criação do projeto, principalmente devido ao tempo limitado para entrega.

As ferramentas auxiliaram na geração inicial de código e em sugestões de implementação, enquanto toda a integração, adaptação, testes e evolução das funcionalidades foram realizadas durante o desenvolvimento do projeto.

O objetivo foi utilizar a IA como ferramenta de produtividade, compreendendo e ajustando o código gerado conforme as necessidades da aplicação.
